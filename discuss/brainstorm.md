# TalentDrop — AI 驱动的人才匹配平台

## 灵感来源

Date Drop 在斯坦福爆火的核心逻辑：
- **深度画像**（66维问卷）替代浅层"左右滑动"
- **每周空投**制造稀缺感和仪式感
- **双向匹配**确保双方都有意愿
- **真实反馈**持续训练算法
- **社区推荐**（Play Cupid）引入社交信任

## 传统招聘的痛点

| 痛点 | 求职者视角 | 企业视角 |
|------|-----------|---------|
| 海投低效 | 投100份简历，回复不到5% | 收1000份简历，合格不到3% |
| 匹配表面化 | 关键词匹配 ≠ 真正合适 | JD写得好 ≠ 岗位真实面貌 |
| 文化盲区 | 入职才知道"水土不服" | 试用期离职成本巨大 |
| 信息不对称 | 薪资、团队氛围全靠猜 | 候选人真实能力难评估 |
| 体验疲劳 | 无限刷岗位，精力耗尽 | 反复面试，效率低下 |

## 核心理念：从"海投海筛"到"精准空投"

**一句话定位：** 每周为你空投最匹配的职业机会/人才，让招聘回归"双向奔赴"。

## 产品机制设计

### 1. Career DNA 深度画像（对标 Date Drop 66维问卷）

**求职者侧 — 50+ 维度深度问卷：**

- **核心价值观**：自主性 vs 协作、稳定 vs 冒险、使命驱动 vs 收入导向
- **工作风格**：远程/混合/到岗偏好、快节奏 vs 深度思考、独立作战 vs 团队协作
- **成长诉求**：5年目标、想要的导师类型、技能提升方向
- **文化偏好**：扁平 vs 层级、加班文化态度、决策方式偏好
- **硬技能**：技术栈、领域经验、项目经历
- **软技能**：沟通风格、冲突处理方式、领导力特质
- **AI 语音分析**：通过语音回答分析沟通能力和性格特质
- **Dream Role 描述**：用自己的话描述理想工作（开放式）

**企业侧 — Company DNA 画像：**

- **团队文化**：真实工作氛围（非 HR 话术）
- **管理风格**：决策流程、汇报关系、自主度
- **成长路径**：晋升机制、学习资源、导师制度
- **团队个性**：现有团队成员的性格分布
- **技术环境**：技术栈、工具链、开发流程
- **真实期望**：这个岗位真正需要什么样的人

### 2. Weekly Talent Drop（每周人才空投）

- **空投时间**：每周二晚 9:00（制造仪式感）
- **求职者**：收到 1-3 个精准匹配的岗位，附带"为什么你们适合"的详细报告
- **企业**：每个岗位收到 3-5 个预筛候选人，附带匹配度分析
- **48小时窗口**：双方必须在窗口期内做出 Accept / Pass 决定
- **稀缺性设计**：每周有限匹配数，迫使双方认真对待每一个匹配

### 3. Mutual Match 双向奔赴

- 求职者可以秘密标记"Dream Company"列表
- 企业可以秘密标记"Dream Candidate"列表
- 如果双方都在对方的列表中 → 系统直接宣布匹配成功
- 跳过所有流程，直接建立联系

### 4. Play Recruiter 伯乐模式（对标 Play Cupid）

- 在职员工可以为朋友/前同事推荐岗位
- 推荐时需回答："为什么 TA 适合这个团队？"
- 推荐成功入职 → 推荐人获得奖励
- 引入社交信任，降低信息不对称

### 5. Team Chemistry 团队化学反应

- 不只匹配公司，更匹配具体团队
- 分析候选人与目标团队成员的性格互补度
- 预测加入后的团队动态变化
- 确保"人岗匹配"升级为"人团匹配"

### 6. 90-Day Guarantee 入职保障

- 基于真实入职后的留存率、满意度数据训练算法
- 算法目标函数 = 90天后双方满意度（非仅仅入职转化率）
- 持续追踪匹配质量，反哺模型迭代

## 用户旅程

### 求职者旅程
```
注册 → Career DNA 深度画像（30min）→ 每周二等待空投
→ 查看匹配报告 → Accept/Pass → 双方 Accept → 直接对话
→ 面试 → 入职 → 90天反馈
```

### 企业旅程
```
注册 → Company DNA 画像 → 发布岗位 + 岗位DNA
→ 每周二查看匹配候选人 → Accept/Pass → 双方 Accept
→ 安排面试 → 发 Offer → 90天追踪
```

## 商业模式

- **求职者**：完全免费
- **企业基础版**：免费（每周 1-2 个匹配）
- **企业专业版**：按月订阅（更多匹配 + 高级分析）
- **成功佣金**：入职成功收取年薪的 8-12%（远低于猎头的 20-25%）
- **Play Recruiter 奖金**：推荐成功，推荐人和平台分成

## 差异化竞争力

| 维度 | 传统招聘平台 | TalentDrop |
|------|-------------|------------|
| 匹配方式 | 关键词/JD 匹配 | 50+维度深度 AI 匹配 |
| 信息量 | 简历 + JD | 价值观 + 文化 + 技能 + 语音 |
| 用户体验 | 无限浏览/投递 | 每周精准空投 |
| 匹配方向 | 单向投递 | 双向奔赴 |
| 反馈闭环 | 无 | 90天真实结果训练 |
| 社交信任 | 无 | Play Recruiter 推荐 |
| 匹配粒度 | 公司级 | 团队级 |

## 初期目标市场

- **Phase 1**：顶尖科技公司 + 顶尖高校应届生（对标 Date Drop 校园策略）
- **Phase 2**：扩展到中高端技术岗位
- **Phase 3**：全行业覆盖

## 核心指标

- 匹配接受率（双方 Accept Rate）
- 面试转化率（Match → Interview）
- Offer 率（Interview → Offer）
- 90天留存率（核心指标）
- 用户满意度 NPS
